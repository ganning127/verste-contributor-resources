MLA Citation Makers

These resources can be used to create MLA citations of resources that you used when simplifying your research paper!

  • Citation Machine

    Citation machine is a free resource that generates MLA citations from information that you provide about a website! Citation Machine automatically alphabetizes the order of the citations to be standard with Verste's guidelines.

  • EasyBib

    Similar to Citation Machine, EasyBib is also a Chegg service that allows users to create MLA citations. All you need to do is enter a website URL and most fields will auto-populate with the necessary information.
